# IPL Player Performance Data Analysis

An end-to-end, in-depth data analysis project exploring all-time and recent IPL player performance using Python and Pandas.
The analysis includes, all-time as well as recent, player-by-player statistical information, player comparisons, custom-made reliability and impact metrics for more meaningful analysis beyond just numbers and statistics.

## Objectives

- Analyze IPL Player batting statistics
- Analyze IPL Player bowling statistics
- Study player performance across all seasons.
- Study recent (last 3 years) vs all time career statistics
- Create visualizations and frontend behaviour (currently using streamlit)
- Build user-readable and reliable analysis metrics

> **Note:** The current dataset covers IPL seasons from **2008–2025** (2026 not included).
